本篇记录了在 Mac OS X EI Capitan 10.11 系统上，搭建 Java 开发环境的步骤。

安装 Java 1.8

进入 Java 下载页面，选择下载 jdk-8u65-macosx-x64 下载链接

  # 命令行查看 Java 安装的情况

  $ java -version
  java version "1.8.0_65"
  Java(TM) SE Runtime Environment (build 1.8.0_65-b17)
  Java HotSpot(TM) 64-Bit Server VM (build 25.65-b01, mixed mode)

安装 Apache Maven 3.3.3

  # 使用 Homebrew 安装 Maven

  $ brew install maven

  # 命令行查看 Maven 安装的情况

  $ mvn -version
  Apache Maven 3.3.3 (0775791599e205a5524ec3e0dfe41d4a06; 2015-04-22T19:57:37+08:00)
  Maven home: /usr/local/Cellar/maven/3.3.3/libexec
  Java version: 1.8.0_65, vendor: Oracle Corporation
  Java home: /Library/Java/JavaVirtualMachines/jdk1.8.0_65.jdk/Contents/Home/jre
  Default locale: zh_CN, platform encoding: UTF-8
  OS name: "mac os x", version: "10.11", arch: "x86_64", family: "mac"

安装 MySQL 5.6

前往 MySQL 下载页，选择下载 Mac OS X 10.10 (x86, 64-bit), DMG Archive 下载链接

安装 Eclipse Mars

打开 Eclipse 下载页面， 下载 Eclipse IDE for Eclipse Committers 4.5.1. Package suited for development of Eclipse itself at Eclipse. Version: Mars.1 Release (4.5.1).

Eclipse 安装 Maven 插件 m2e

Eclipse 默认不支持 Maven。需要给它添加 m2eclipse 插件。
打开 Eclipse，选择 Help，Eclipse Marketplace。 在 All Markets 选项下，搜索 maven。
有很多搜索的结果，请安装 Maven Integration for Eclipse (Luna)。如下图所示
